# YOLOv8 Vehicle Speed Evaluation against VS13 Dataset

## Overview
This project contains a Python evaluation script (`evaluate_speed.py`) designed to calculate the accuracy of a YOLOv8-based vehicle speed estimation model. The script aggregates frame-by-frame speed predictions generated by YOLOv8 and compares them against the established ground truth speeds from the VS13 public benchmark dataset. 

The script calculates standard regression metrics (MAE, MAPE, RMSE) to quantify the model's performance and generates a comparative visual plot.

## Directory Structure

To run the evaluation script successfully, ensure your project directory is structured as follows:

```text
your_project_folder/
├── evaluate_yolo_speed.py       # The main evaluation script
└── vehicle_speeds/              # Subfolder containing the YOLOv8 output files
    ├── vehicle_speeds_1.csv
    ├── vehicle_speeds_2.csv
    ├── ...
    └── vehicle_speeds_12.csv
```


## Prerequisites
Ensure you have Python installed along with the following required libraries:
* pandas
* numpy
* matplotlib
* scikit-learn

You can install these dependencies using pip:
pip install pandas numpy matplotlib scikit-learn

## Usage
1. Open the `evaluate_speed.py` script.
2. Update the `ground_truth_speeds` array at the top of the script with the exact, corresponding speeds for the 12 videos you processed.
3. Open your terminal and navigate to the project directory.
4. Run the script:
python evaluate_speed.py

The script will output a data table in the console showing the estimated median speeds versus the ground truth, followed by the performance metrics. A Matplotlib window will also open displaying a line chart of the results.

## About the VS13 Dataset
This project utilizes the **VS13 Dataset**, a public benchmark specifically compiled to facilitate research in audio-video vehicle speed estimation. 

### Key Dataset Characteristics:
* **Controlled Ground Truth:** The speed of the vehicles in the dataset is maintained perfectly stable by the vehicles' on-board cruise control systems. This provides a highly reliable single "ground truth" speed for each entire video.
* **Vehicle Diversity:** The dataset features 13 different vehicles selected to represent a diverse range of manufacturers, production years, engine types, power outputs, and transmissions. 
* **Speed Range:** The vehicles pass the camera at known, constant speeds ranging from 30 km/h to 105 km/h.
* **Recording Specifications:** The dataset comprises 400 annotated audio-video recordings. The videos are recorded in MP4 format, at full HD resolution, 30 frames per second (fps), and each video is exactly 10 seconds long.
* **Recording Environment:** The footage was recorded on a local road in an urban environment to simulate real field conditions, capturing single drives of single vehicles.

Because the VS13 dataset relies on cruise-controlled, constant speeds, evaluating frame-by-frame YOLOv8 predictions requires aggregating the output (using the median speed across all frames) to compare against the single dataset ground truth value.

**Official Dataset Link:** [http://slobodan.ucg.ac.me/science/vs13/](http://slobodan.ucg.ac.me/science/vs13/)
